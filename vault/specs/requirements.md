# Requirements

## Functional Requirements

- [ ] AI-powered core feature (not just a ChatGPT wrapper)
- [ ] Addresses a real environmental / sustainability challenge
- [ ] Demonstrates measurable or visible impact
- [ ] Working end-to-end demo

## Non-Functional Requirements

- [ ] Clean, well-structured codebase
- [ ] Polished UI/UX
- [ ] Accessible via web (online submission)
- [ ] Performant enough for live demo

## Out of Scope

- Mobile native app
- Production-scale infrastructure
- Paid API tiers (keep costs minimal)
